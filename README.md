# Chimi Party Landing Page

A modern and responsive landing page built for **Chimi Party**, a balloon decoration business based in Turin, Italy.

The website is designed to present the brand, showcase its services, and make customer contact quick and simple through WhatsApp, phone, and social media.

## Preview

This project includes:

- A floating expandable navbar
- A hero section with image carousel
- A services section with interactive cards
- An about section
- A custom footer with contact access and social links
- A contact modal with direct phone numbers

## Built With

- React
- Vite
- JavaScript
- CSS
- React Bootstrap
- React Icons

## Project Goal

The goal of this project is to create a simple, clean, and visually appealing landing page focused on client acquisition for a small event decoration business.

## Features

- Responsive layout for desktop, tablet, and mobile
- Smooth internal navigation with anchored sections
- WhatsApp direct contact button
- Expandable navbar menu
- Service cards with hover interaction on desktop
- Contact modal for direct phone calls
- Clean visual style with soft pastel branding
